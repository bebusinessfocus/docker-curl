# docker-curl

Alpine-based image with just curl

## Usage

\$ docker run --rm bebusinessfocus/curl -fsSL https://bebusinessfocus.com/

## Build

\$ docker build -t local/curl .

## Run (local)

\$ docker run --rm local/curl -fsSL https://bebusinessfocus.com/
